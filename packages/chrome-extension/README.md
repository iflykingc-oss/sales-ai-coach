# 销冠AI教练 - Chrome扩展

实时销售通话辅助工具，支持Zoom、Google Meet、Microsoft Teams。

## 功能特性

### 🎯 实时教练
- 在通话中显示浮动助手窗口
- 提供实时销售建议和异议处理提示
- 检测购买信号并提醒推进成交

### 📊 话轮比追踪
- 实时监控销售与客户的发言比例
- 理想比例：40-45% 销售发言
- 颜色指示：绿色=理想，黄色=过多/过少

### 🛡️ 异议检测
- 自动识别客户异议关键词
- 提供针对性的异议处理建议
- 记录异议次数和类型

### 👤 客户画像
- 显示当前客户的基本信息
- 展示客户性格特点和痛点
- 帮助销售更好地了解客户

## 安装方法

### 开发模式安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `packages/chrome-extension` 目录
5. 扩展安装完成

### 使用方法

1. 安装扩展后，在Zoom/Google Meet/Teams通话页面会自动激活
2. 右下角会出现浮动助手窗口
3. 通话过程中会实时显示：
   - 话轮比（发言比例）
   - 异议检测和处理建议
   - 购买信号提醒
4. 点击扩展图标可以查看详细数据

## 文件结构

```
chrome-extension/
├── manifest.json      # 扩展配置
├── background.js      # 后台服务
├── content.js         # 页面注入脚本
├── content.css        # 注入样式
├── popup.html         # 弹出窗口HTML
├── popup.js           # 弹出窗口脚本
├── sidepanel.html     # 侧边栏HTML
└── icons/             # 扩展图标
```

## 配置选项

在扩展设置中可以配置：

- **自动显示**: 通话开始时自动显示助手
- **提示时长**: 每条提示的显示时长（默认8秒）
- **异议检测**: 开启/关闭异议自动检测
- **购买信号**: 开启/关闭购买信号检测

## 技术实现

- **Manifest V3**: 使用最新的Chrome扩展API
- **Content Script**: 注入到视频通话页面
- **Background Service Worker**: 后台处理和分析
- **Chrome Storage API**: 持久化配置和数据
